
查询（）：
    1. 访问index.jsp页面
    2. index页面发出查询员工列表请求
    3. EmployeeController来接受请求，查出员工数据
    4. 来到list.jsp页面进行展示


查询（使用json的形式）：
    1. 在index.jsp页面直接发送ajax请求进行员工分页数据的查询
    2. 服务器将查出的数据信息，以json字符串的形式返回给浏览器
    3. 浏览器收到json字符串数据信息，可以使用js对json进行解析，使用js通过dom对象
        进行增删改查的方式改变页面数据
    4. 返回json，实现客户端的无关性



新增员工：逻辑
    1. 在index.jsp页面点击新增
    2. 弹出对话框
    3. 去数据库查询部门列表，显示在对话框中
    4. 用户输入数据信息，并进行校验：
           jQuery的前端校验，ajax用户名重复校验，重要的数据都需要加上(后端校验（使用springmvc自带的JSR303）,唯一约束)
    5. 完成保存的工作
